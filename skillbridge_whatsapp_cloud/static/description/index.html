Frequently Asked Questions

Does it support Odoo Enterprise?
Yes — SkillBridge is fully compatible with Odoo Enterprise and integrates with Enterprise modules and access control.
Does it support Odoo Community?
Yes — Core messaging and campaign features work with Odoo Community installs; some integrations with Enterprise-only modules may vary.
Does it require Meta verification?
You must configure a Meta Business account and WhatsApp Business Account. SkillBridge works with the official Meta Cloud API and supports the verification flows.
Can I send PDFs?
Yes — Attach PDFs, images and other supported media types to messages and campaigns.
Can I use multiple WhatsApp numbers?
Yes — Multi-company support allows separate credentials and WABA numbers per company.
Does it support media attachments?
Yes — Images, documents and other supported media are uploadable and trackable per message.
Are message statuses tracked?
Yes — Delivery and read events are logged and available in message logs and analytics.
