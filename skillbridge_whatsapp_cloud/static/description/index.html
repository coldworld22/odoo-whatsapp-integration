Frequently Asked Questions

Does it support Odoo Enterprise?
Yes — SkillBridge is fully compatible with Odoo Enterprise and integrates with Enterprise modules and access control.
Does it support Odoo Community?
Yes — Core messaging and campaign features work with Odoo Community installs; some integrations with Enterprise-only modules may vary.
Does it require Meta Business Verification?
You must configure a Meta Business account and WhatsApp Business Account. SkillBridge works with the official Meta Cloud API and supports verification and approval workflows.
Can I send invoices as PDFs?
Yes — Attach invoice PDFs to messages; invoices shared over WhatsApp are tracked in message logs and can be linked back to the Odoo invoice record.
Can I use multiple WhatsApp numbers?
Yes — Multi-company support allows separate WABA numbers and credentials per company, each with individual templates and settings.
Does it support media attachments?
Yes — Images, PDFs and other supported media types can be attached and uploaded to Meta for delivery and tracking.
Can I track message delivery and read status?
Yes — Delivery and read events are captured from Meta Cloud and logged for each message, visible in message logs and analytics.
Does it support campaigns?
Yes — The Campaign Manager supports batch sends, throttles, retry policies, and staged drip campaigns with per-company configuration.
