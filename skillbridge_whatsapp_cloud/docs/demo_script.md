# WhatsApp Cloud Messaging - 90 Second Demo Script

## Goal
Show how a user connects Meta, sends a Sales Order message, and sees delivery logs in under 90 seconds.

## Script
1) "This is WhatsApp Cloud Messaging for Odoo 17. It uses the official WhatsApp Cloud API - no extra gateway."
2) Open Settings - "We add the Meta token, phone number ID, verify token, and app secret here."
3) Open a Sales Order - "Click Send WhatsApp to choose text, template, or interactive mode."
4) Select a template and attach the Sales Order PDF - "Variables and compliance hints are shown in the wizard."
5) Send - "Messages are logged immediately."
6) Open WhatsApp Logs - "You can track sent, delivered, read, and failed with pivot KPIs."
7) Optional: show Campaigns - "Queue based campaigns with throttling and retries."
8) Close - "Built for Sales and Accounting teams that need fast, compliant customer updates."

## Tips
- Keep the video under 2 minutes.
- Highlight opt-in enforcement and STOP/START keyword handling.
- End with a screenshot of the logs and your support email.
